渐变

自顶向下渐变：

background-image: -webkit-linear-gradient(to top,red,blue);
background-image: linear-gradient(to bottom,red,blue);
/* -webkit- 的第一个参数是起始方向，而W3C标准语法第一个参数是目的方向*/

左上到右下渐变：

background-image: -webkit-linear-gradient(top left,red,blue);
background-image: linear-gradient(top bottom right,red,blue);

使用角度

background-image: -webkit-linear-gradient(90deg,#f00,#00f);
background-image: linear-gradient(90deg,#f00,#00f);
//webkit的方法和W3C 标准方法的角度参数的意义不同
//W3C 中0deg表示向上变化，90deg表示向右变化，其实可以参考钟表的指针，0点表示0度，
//方向向上，表示从下向上渐变，而3点表示90度，方向指向右，表示从左向右渐变。

//webkit 中这个方向便是坐标系的方向，0deg 指向右边，表示从左到右渐变，其他以此类推。

多种颜色渐变：

background-image: -webkit-linear-gradient(left,red,yellow,blue,green,red);
background-image: linear-gradient(to right,red,yellow,blue,green,red);

自定义渐变宽度:

background-image: -webkit-linear-gradient(left,
	red 0%,
	green 80%,
	blue 100%);
background-image: linear-gradient(to right,
	red 0%,
	green 80%,
	blue 100%);

径向渐变：

background-image: radial-gradient(red , yellow);

圆形渐变：

background-image: radial-gradient(circle,red 0%,yellow 10%,green 20% ,blue 30%);

椭圆渐变：

background-image: radial-gradient(ellipse,red 0%,yellow 10%,green 20%,blue 30%);

自定义半径：

background-image: radial-gradient(200px 20px,green,red);

自定义半径和位置

background-image: radial-gradient(200px 80px at center bottom,yellow 70%,red 100%);

设置渐变算法：

background-image: radial-gradient(farthest-side ellipse at center bottom,yellow 70%,red 100%);	
// farthest-side : 中心距离最远的边决定半径
// farthest-corner：最远的顶点
// closest-side：最近的边
// closest-corner：最近的顶点

水平重复渐变：

background-image: repeating-linear-gradient(red,green 40px,orange 15px);

径向重复渐变：

background-image: repeating-radial-gradient(red 20px,yellow 30px,blue 40px);

利用渐变制作作业本效果

background-image: repeating-linear-gradient(to top,#f9f9f9 0px,#fff 29px,#000 30px); 
background-size: 100% 30px;
background-repeat:repeat;
// 在第30个像素处突变颜色，产生一条横线,然后重复这部分背景图片

利用渐变制作纹理背景

background-color: #282828;
background-image: radial-gradient(black 15%,transparent 16%),
		radial-gradient(black 15%,transparent 16%),
		radial-gradient(rgba(255,255,255,0.1)15%,transparent 20%),
		radial-gradient(rgba(255,255,255,0.1)15%,transparent 20%);
background-position: 0 0,8px 8px,0 1px,8px 9px;
background-size: 16px 16px;
